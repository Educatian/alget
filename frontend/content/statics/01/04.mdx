## Static vs. Kinetic Friction

| Property | Static Friction ($F_s$) | Kinetic Friction ($F_k$) |
|----------|------------------------|-------------------------|
| When | Object at rest | Object moving |
| Range | $0 \leq F_s \leq \mu_s N$ | $F_k = \mu_k N$ (constant) |
| Coefficient | $\mu_s$ (higher) | $\mu_k$ (lower) |
| Key point | **Variable** up to max | **Fixed** value |

### The Critical Relationship

$$
\boxed{F_s \leq \mu_s N}
$$

Static friction **adjusts** to match the applied forceâ€”until it reaches its maximum.

$$
(F_s)_{\max} = \mu_s N
$$

> âš ï¸ **Common Pitfall**: Don't assume $F_s = \mu_s N$ always! The equality holds **only** at impending motion.

## Impending Motion

**Impending motion** is the condition when an object is *just about to slip*.

At impending motion:
- $F_s = (F_s)_{\max} = \mu_s N$
- The block is still stationary, but any additional force would cause slipping

### Sign of Impending Motion

Before solving, you must **assume a direction** of impending motion:
1. Guess which way the object tends to slide
2. Draw friction opposing that direction
3. Solve and check: if $P > 0$, your assumption was correct
